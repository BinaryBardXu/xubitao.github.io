骚窝第一年目标

Apr 18, 2016 | 騷窩我的窩 | Hits

第一年的两个目标。

试用期目标：技术进步－以工程实践为基础，培养良好习惯和执行力

  • Git
    四月完成学习，五月达到熟练应用。

  • Jenkins
    进阶学习，熟练掌握Pipeline的用法。

  • Linux
    熟练掌握日常的Linux应用。每周学习两章，完成《鸟哥的Linux私房菜》的第一轮的学习。

  • Docker
    最迟六月学习。理解容器原理，掌握容器基本操作，能够通过容器发布应用。

  • BDD
    四月底开始学习，两月内达到熟练应用。

  • DevOps
    理解DevOps原理，了解可落地的措施。

  • 学习一门新语言
    八月起学习Scala.

  • Session
    完成四次session，此前在上海已经有一次，后面再完成三次，五个礼拜一次。

  • 博客
    每周至少一篇博客筆記或文章。

  • 英语
    提高英语口语和听力，每天半小时观看、琢磨剧本。《老友记》第一季无障碍listen。

  • 阅读
    每周阅读一本人文社科书籍。

  • 转正
    再艰难困苦也要努力转正。

转正半年内目标：过程改进－整合敏捷方法，培养过程改进洞见力

  • Kanban
    结合在苏宁的尝试，进一步探索看板方法在团队中的应用。

  • Scrum
    精准理解Scrum关键原理、要素。

  • 精益
    此前尝试过看板、Scrum，唯独没有实践过精益。理清精益理念及与敏捷的关系。

  • DevOps
    深刻理解DevOps在企业中的应用。
